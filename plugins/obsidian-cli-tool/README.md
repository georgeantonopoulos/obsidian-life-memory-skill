# obsidian-cli-tool

First-class OpenClaw plugin tools for Obsidian vault access backed by `obsidian-cli`.

## Tools
- `obsidian_read`
- `obsidian_search`
- `obsidian_search_content`
- `obsidian_create`
- `obsidian_append`
- `obsidian_move`
- `obsidian_list`
- `obsidian_daily_read`
- `obsidian_daily_append`

## Purpose
This lives inside `obsidian-life-memory-skill` because it is the hard-capability complement to the skill's existing guidance: use `obsidian-cli` for vault operations and avoid raw file tools on vault notes.
