# Life Memory Patterns

## Daily Note

Path: `Daily/YYYY-MM-DD.md`

```markdown
---
type: daily-log
created: 2026-02-15
tags: [daily, log]
---

# 2026-02-15

## Events
- **09:10** [Planning] Finalized school shortlist [[Athens Move]] [[Sxoli Moraiti]]
- **13:40** [Health] Appointment confirmed [[Maya]] [[Doctor Follow-up]]

## Connectors
- [[SOUL]]
- [[USER]]
- [[LIFE_CONTEXT]]
```

## Long-Term Memory Pattern

Path: `MEMORY.md`

```markdown
## Distilled 2026-02-15
- Source: [[Daily/2026-02-15]]
- Updated: 2026-02-15 21:00
- **09:10** [Planning] Finalized school shortlist [[Athens Move]] [[Sxoli Moraiti]]
```

## Graph Hygiene

- Every new event should link to at least one stable hub note.
- Prefer reusable entities (`[[Athens Move]]`, `[[Maya]]`) over plain text.
- Keep internal tool clutter out of the graph using `.obsidianignore`.
